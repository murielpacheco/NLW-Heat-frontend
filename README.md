# NLW Heat - Rocketseat

## DoWhile 2021

<p>Projeto desenvolvido durante o evento NLW Heat promovido pela Rocketseat.
A aplicação foi desenvolvida com as seguintes tecnologias e ferramentas:</p>

<h2>Frontend</h2>
  <li> React JS
  <li> Vite JS (template do projeto "react-ts")
  <li> Typescript
  <li> SCSS 
  <li> Socket-io

<h2>Back end</h2>
  <li> Node
  <li> Typescript
  <li> SQLite
  <li> Prisma
  <li> Socket-io
